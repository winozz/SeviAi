## 📂 Directory Details

### `/api` - REST API & Machine Learning (3 files)
Purpose: Run the chatbot and inference

**Files**:
- `app.py` - FastAPI server with 15+ endpoints
- `hybrid_chatbot.py` - Hybrid NB+NN classifier
- `logger.py` - Chat logging system
- `__init__.py` - Python package marker
- `README.md` - API documentation

**Use**: Start API here
```bash
python -m uvicorn api.app:app --port 8000
```

---

### `/training` - Model Development (8 files)
Purpose: Train, test, and improve the ML model

**Files**:
- `train_naive_bayes.py` - Train Naive Bayes (8s)
- `train_hybrid.py` - Train Neural Network
- `test_intents.py` - Quick evaluation (5-30s)
- `expand_intents.py` - Auto pattern generation (3s)
- `api_stress_test.py` - Async stress test (2-5m)
- `automated_training.py` - Full pipeline (2m)
- `test_chatbot.py` - Model validation (30s)
- `test_api.py` - API testing
- `README.md` - Training guide

**Use**: Improve model accuracy
```bash
python training/test_intents.py 8000 5
```

---

### `/data` - Training Data (1 file)
Purpose: Intent definitions and patterns

**Files**:
- `CvSU_intents.json` - 227 patterns, 23 intents

**Format**:
```json
{
  "intents": [
    {
      "tag": "admissions_requirements",
      "patterns": ["What are requirements?", ...],
      "responses": ["For admission you need..."]
    }
  ]
}
```

**Use**: Add patterns here, then retrain
```bash
# Edit CvSU_intents.json
# Then retrain:
python training/train_naive_bayes.py
```
